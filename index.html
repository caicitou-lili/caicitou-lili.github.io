历史的镜子：1926—1930 与 2026—2030，我们站在哪里？

“历史不会简单重复，但它押着相同的韵脚。”——马克·吐温

引言：两个时代的奇异共鸣

每一代人都觉得自己活在史无前例的时代。但历史学家知道，人类的剧本反复循环——技术革命带来财富幻觉，财富幻觉滋养政治极端，政治极端催生贸易对抗，贸易对抗击碎繁荣，繁荣崩塌之后才有真正的重建。

今天，当我们把 2024—2026 年的世界与 1926—1929 年并排摆放，那种令人不安的相似性便会扑面而来：同样的股市狂欢，同样的技术乐观主义，同样的民粹政治崛起，同样的贸易保护主义回潮。

当然，历史不是命运。这个世界也有 1926 年不曾拥有的东西——央行工具箱、社会安全网、全球治理框架，以及那段历史留下的惨烈教训。但教训需要被记得，才能发挥作用。

Claude Code 动态工作流 — 为每个任务定制专属引擎

上周，Claude Code 发布了动态工作流（Dynamic Workflows）功能。Claude 现在可以在运行中动态编写自己的编排脚本，为手头的任务定制专属的执行引擎。

虽然默认的 Claude Code 是为编程构建的，但事实证明很多任务都类似于编程任务。然而，有些特定类型的任务需要在 Claude Code 之上构建定制引擎才能达到最佳效果——比如深度研究、安全分析、Agent 团队协作、代码审查。

Workflows 让你可以动态创建这些引擎，让 Claude 原生解决以上所有问题，还能在团队间分享和复用。

如何用 Obsidian+Claude 搭建专属AI第二大脑

绝大多数人都在做无效笔记：

随手高亮文章、堆叠书签、认真记录会议纪要、灵感一闪即刻记下。

但所有这些笔记，最后的归宿只有一个：被遗忘的数字坟场

躺在尘封的文件夹里，再也不会打开、不会复盘、不会复用。

记笔记的本质从来不是存储，而是复用与思考。

这也是 Obsidian 突破 150万+ 付费用户 的核心原因：它打败了所有主流笔记软件，靠的不是记录功能，而是AI 深度集成能力。

当你把 Claude 接入你的 Obsidian 知识库，真正的蜕变就此发生：

你的笔记不再是静态的文字文件，而是一套可被 AI 读取、检索、推理、迭代、进化的私人思维系统。

你向 Claude 提问，它不用互联网泛知识作答。它调用你的笔记、你的研究、你的灵感、你的专属上下文，给你定制化答案。

这才是真正的第二大脑。 不是堆满废弃文件的文件夹，而是一个陪你思考、帮你复盘、替你迭代、持续进化的智能伙伴。

零基础、零技术门槛，全文手把手教学，普通人今天就能落地。
